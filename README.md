# Student Stress vs Study Hours Analysis

## Overview
This project explores how study hours and sleep patterns relate to student stress levels.

The goal was to understand how data can help reveal patterns in student learning and wellbeing.

## Data Collection
Data was collected through a student survey including:
- Study hours per day
- Sleep hours
- Self-reported stress level (1–10)

## Tools Used
- Google Forms
- Excel / Google Sheets
- Data visualization

## Key Insights
- Students studying more than 6 hours daily reported higher stress levels.
- Lower sleep duration was associated with higher stress scores.
